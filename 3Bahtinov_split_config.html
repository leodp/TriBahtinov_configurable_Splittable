Tri Bahtinov Mask, splittable, configurable — v1.6

Tri-Bahtinov mask generator, based on Satoru Takagi implementation. The mask can be split, to allow bigger masks to be 3D printed at home. Slits can be rotated of an extra offset angle. Programmed by Leonardo Della Pietra leodp@yahoo.com

The SVG 2D vector file contains all the paths needed to build the mask. For further info please refer to external resources; a possible search could be done with the text “Turning an SVG into a 3D Printed Object with FreeCAD”.

The STL 3D mesh may have some issues like open faces and free-standing faces not enclosing the mask internal volume. Try to clean them in a SW, like for example FreeCAD. At best you can select the “Mesh” workbench and export them (again) as a mesh. If you’re lucky it works. Otherwise you may want to try the mesh tool “Fill holes”, increasing the max number of edges to 4. YMMV

FFT half-view size is the half-width shown from the FI center in the selected units. Defocus uses waves referenced to 530 nm (green); in RGB mode each channel scales physically from this green reference.
Practical parameter guide:
1) For clearer spikes: use logarithmic or exponential-lift mapping, and decrease display gain (for example 0.0001 to 0.01).
2) For dense/thin slits (high Bahtinov factor): keep grid at 2048 or 4096 and raise mask resolution boost to 2x or 4x.
3) If spikes look clipped or too close to center: increase FFT half-view size (for example 3 to 6).
4) If the aperture looks too coarse: reduce aperture margin factor toward 1 and/or increase resolution boost.
