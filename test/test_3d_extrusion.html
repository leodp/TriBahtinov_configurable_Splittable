3D Extrusion Test – 5 Approaches

Test shape: A circular disk (R=50) with a rectangular hole (40×10 centered at origin) and a small circular hole (R=8 at x=25, y=25). Thickness=3.
Each approach generates an ASCII STL. Download and evaluate in 3D software (e.g. PrusaSlicer, MeshLab, FreeCAD).

Method 1: Bridge-Edge Merge (current approach)

Merge all holes into the outer polygon by inserting bridge edges (zero-width corridors). The result is a single simple polygon. Ear-clip triangulate it. Extrude top/bottom + side walls from the merged boundary. Bridge edges create doubled-back vertices that form zero-width passages from outer to each hole.

Method 2: Separate Face Triangulation (no bridges)

Triangulate each face independently: outer polygon minus holes using a fan-based decomposition. The outer boundary, each hole boundary, and bridge connections are all separate triangle fans. Top face = outer triangles with holes subtracted via constrained fan decomposition. Side walls generated separately for outer and each hole boundary.

Method 3: Radial Fan Triangulation

For the top/bottom faces: connect every boundary vertex (outer + holes) to a centroid or sweep-line via radial fans from the centroid. Holes are cut by reversing winding within their span. Side walls from each boundary loop independently.

Method 4: Grid/Scanline Boolean

Overlay a fine grid on the bounding box. For each grid cell, test whether its center is inside the outer polygon and outside all holes. Emit two triangles per valid cell for top/bottom faces. Side walls are generated by detecting boundary cells. Approximation depends on grid resolution.

Method 5: Bridge-Edge Merge with Duplicate Vertex Offset

Similar to Method 1, but bridge edges are given a tiny perpendicular offset (epsilon = 0.01) so the bridge passage is a real (very thin) physical corridor instead of zero-width. This avoids degenerate zero-area triangles along bridges. The merged polygon is then ear-clipped and extruded.

Method 6: Monotone Polygon Decomposition

Split the polygon-with-holes into y-monotone pieces using a sweep-line algorithm, then triangulate each monotone piece. This avoids bridge edges entirely and handles holes natively. Side walls from each original boundary loop.

Method 7: Separate Solids per Region

Instead of one merged polygon, treat the outer shell and each hole as separate extrusion operations. The outer disk is a solid cylinder. Each hole is a separate solid cylinder. The STL contains the outer solid plus all hole solids with inverted normals (subtractive). Some software interprets overlapping reversed-normal solids as boolean subtraction.
