3D Extrusion Test v2 – Refined Methods

Test shape: circular disk (R=50) with a rectangular hole (40×10) and a circular hole (R=8 at 25,25). Thickness=3.
Focused on fixing Method 4's boundary resolution while keeping its correct boolean logic.

Key insight: Method 4's boolean logic (even-odd fill) is topologically correct. The fix: replace grid cells with exact polygon edge intersections → Scanline Exact Triangulation.

Side wall normal bug found: sideWalls() already produces correct outward-facing normals for both CCW outer boundaries and CW hole boundaries. Methods 2 & 6 incorrectly used sideWallsReversed() for holes, inverting their normals.

Method A: Scanline Exact Triangulation ★ (Recommended)

Same boolean logic as grid Method 4 (even-odd fill rule), but uses exact polygon edge intersections instead of grid cells. All original polygon vertices are preserved in the output mesh. Boundaries are exact polygons, not staircase approximations.
Algorithm: Collect all vertex y-values → define horizontal slabs. In each slab, compute exact x-intersections of all active edges. Sort by x, pair by even-odd rule → solid spans become trapezoids (2 triangles each). Side walls from original boundary loops.

Method B: Scanline Adaptive (Refined A)

Same as Method A but subdivides tall slabs (height > 3mm) into smaller strips. This avoids very elongated thin triangles and improves mesh quality for slicers. Boundary geometry is still exact (subdivided edges are linear interpolations of the same polygon edges).

Method C: Bridge Merge + Fixed Normals

Same bridge-edge merge as v1 Method 1 for face triangulation, but with corrected side walls: uses sideWalls() for both outer and holes (fixing the normal inversion bug). This tests whether the bridge approach works once normals are correct, or if bridge seams are inherently problematic.

Method D: Bridge Merge + Epsilon Offset + Fixed Normals

Bridge-edge merge with epsilon-width corridors (0.01 mm) instead of zero-width. This avoids degenerate zero-area triangles. Combined with fixed normals. Tests whether epsilon bridges solve the manifold issue.

Method E: Fine Grid Boolean (0.25mm)

Same as v1 Method 4 but with 4× finer grid (0.25mm cells instead of 1mm). For comparison: shows that even at higher resolution, staircase artifacts persist. File is much larger. Demonstrates why the scanline approach is fundamentally better.
